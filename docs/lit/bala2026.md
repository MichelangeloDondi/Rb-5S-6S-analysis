# bala2026

Isotope-dependence-of-collisional-width framework: why beta\_85=beta\_87 is expected (thermal impact, tiny reduced-mass difference).

arXiv:2605.01908 (Torun group, Julienne/Ciurylo lineage). Gives the theoretical isotope-dependence of collisional widths/shifts from reduced mass + C6 + scattering length. In the thermal impact regime the 85/87 reduced-mass difference is ~1%, so a negligible width isotope-effect is expected — our measured beta_85 = beta_87 null is the physically expected result, not a surprise. Their Hg-Rb ultracold system is a different regime; cite the framework, not the numbers.
